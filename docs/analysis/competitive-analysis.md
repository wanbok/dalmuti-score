## 1. Executive Summary

보드게임 점수 추적 시장은 **네이티브 모바일 앱** 중심으로 형성되어 있으며, BG Stats, Scory, Score Anything 등이 대표적이다. 웹 기반 경쟁자로는 KeepTheScore.com이 유일하다.

**핵심 발견**: 모든 경쟁앱은 **포인트 증감(+/-)** 방식의 점수 입력을 사용한다. **순위 기반 드래그앤드롭 입력**을 제공하는 앱은 존재하지 않는다. 이것이 달무티 점수판의 가장 강력한 차별점이다.

**시장 기회**:
- 달무티 전용 게임 특화 기능 (역할명, 세금 규칙 등)
- 통계/분석 기능 (승률, 점수 추이 그래프)
- UX 폴리시 (다크 모드, 테마, 애니메이션)
- PWA를 통한 네이티브 앱 수준의 접근성

**주요 위험**: 통계 기능 깊이, 오프라인 안정성, 데이터 내보내기에서 성숙한 네이티브 앱 대비 열위.

## 4. 상세 경쟁앱 분석

### 4.1 BG Stats (Board Game Stats)

**개요**: 보드게임 추적 분야의 표준. 2500개 이상의 게임별 스코어시트와 BGG 완전 연동.

**강점**:
- 업계 최고 수준의 통계 (H-index, 승률, 플레이 시간 차트, Head-to-Head)
- 게임별 커스텀 스코어시트 (달무티 포함 가능성)
- 16개 언어 지원, 클라우드 동기화
- 확장팩/협동 게임/팀 플레이 추적

**약점**:
- 캐주얼 사용자에게 과도한 복잡성
- 네이티브 앱 설치 필수 (웹 버전 없음)
- 순위 기반 입력 미지원 — 포인트 수동 입력만 가능
- 클라우드 동기화 유료 ($0.99/년)

**통계 기능 심층**:
- 승률 계산: 전체/게임별/상대별
- 점수 추이: 시간별 그래프, 최고/최저/평균 점수
- H-index: 플레이 다양성 지표
- 상대별 비교: 특정 상대와의 전적

**시사점**: 통계 기능의 벤치마크. 모든 기능을 복제할 필요 없이 달무티에 가장 의미 있는 지표(승률, 평균 순위, 상대 전적)를 선별 구현.

---

### 4.2 Scory — Score Keeper

**개요**: 게이머가 만든 올인원 게임 나이트 도구. Quick Score + Sheet Score 듀얼 모드.

**강점**:
- 유틸리티 도구 통합 (주사위, 타이머, 랜덤 플레이어 선택기)
- Sheet Score 모드: 라운드별 기록에 적합
- 야찌 전용 모드로 게임 특화 가능성 입증
- 자동 저장, 빠른 인터페이스

**약점**:
- 통계/분석 기능 없음
- 순위 기반 입력 미지원
- 다크 모드/테마 커스터마이징 없음
- 클라우드 동기화 없음

**게임플레이 확장 심층**:
- 야찌 모드: 게임별 커스텀 UI의 성공 사례
- 주사위: 1-6면, 다중 주사위, 커스텀 면
- 타이머: 라운드 타이머, 턴 타이머

**시사점**: 유틸리티 도구(주사위, 타이머)는 달무티와 직접적 관련 없으나, **야찌 전용 모드가 게임 특화 UI의 성공 사례**. 달무티 전용 기능(역할명, 세금 규칙)도 이와 유사한 접근 가능.

---

### 4.3 Score Anything

**개요**: 광고 없는 깔끔한 점수 카운터. 디자인과 사용성에 집중.

**강점**:
- **다크 모드**: OLED 배터리 절약, 야간 게임에 최적
- **24색 플레이어 커스터마이징**: 개인화 UX의 좋은 사례
- 원탭 점수 입력, 실행 취소 지원
- 완전 무료 + 광고 없음 (높은 사용자 신뢰)

**약점**:
- 멀티라운드 누적 점수 미지원
- 통계/분석 없음
- 순위 기반 입력 미지원
- 클라우드 동기화 없음

**UX 심층 분석**:
- 다크 모드 구현: 시스템 설정 연동 + 수동 토글
- 색상 테마: 24개 미리 정의된 색상, 플레이어별 할당
- 점수 회전: 각 플레이어를 향해 점수가 회전되는 UX (대면 게임 시 유용)

**시사점**: **다크 모드 + 플레이어 색상 커스터마이징은 우선 구현 대상**. "광고 없음" 웹앱의 장점을 더 강조할 수 있음. 점수 회전 UX는 달무티에서는 불필요.

---

### 4.4 Keep Score GameKeeper

**개요**: 파티 게임 중심의 소셜 점수 관리 앱.

**강점**:
- 크로스 디바이스 동기화 (계정 기반)
- 소셜 기능: 버저, 랜덤 플레이어 선택기
- 카운트다운 타이머
- 무제한 플레이어 수

**약점**:
- Freemium 모델 (무료 1게임, 유료 $0.99-$3.99)
- 순위 기반 입력 미지원
- 통계/분석 없음
- 다크 모드 없음

**게임플레이 심층**:
- 타이머: 카운트다운 방식, 커스텀 시간 설정
- 버저: 게임 이벤트 알림용 (달무티 라운드 종료 시 활용 가능)
- 랜덤 선택: 첫 번째 플레이어 결정용

**시사점**: 크로스 디바이스 동기화는 중요한 기능 격차. 우리는 계정 기반 대신 **URL 공유 방식**으로 해결 가능. 타이머는 하우스 룰(턴 제한 시간) 지원에 활용 가능.

---

### 4.5 KeepTheScore.com (가장 직접적인 경쟁자)

**개요**: 웹 기반 스코어보드 생성 플랫폼. 우리와 동일한 플랫폼(웹)을 사용하는 유일한 주요 경쟁자.

**강점**:
- **웹 기반** — 설치 불필요, URL로 즉시 접근
- 실시간 멀티 디바이스 동기화 (URL 공유)
- 스트리밍 통합 (OBS, TV 디스플레이)
- 풀스크린 스코어보드 모드
- 다크 모드, 로고/색상 커스터마이징
- 200명까지 무료 지원

**약점**:
- 범용 플랫폼 (카드 게임 최적화 아님)
- 순위 기반 입력 미지원
- 무료 티어에 광고 포함
- 스포츠/이벤트 중심 설계
- 게임별 커스텀 규칙 없음

**UX/기능 심층**:
- 다크 모드: CSS 변수 기반, 토글 스위치
- 공유: 고유 URL로 읽기 전용/편집 모드 분리
- 풀스크린: TV 디스플레이용 확대 모드
- 애니메이션: 점수 변경 시 카운트업 애니메이션

**시사점**: 플랫폼 측면에서 가장 직접적 경쟁자이나 **타겟 사용자가 다름** (라이브 이벤트/스포츠 vs 카드 게임). 참고할 기능: **URL 공유, 풀스크린 모드, 점수 변경 애니메이션**.

---

### 4.6 Scored — Board Game Tracker

**개요**: ELO 랭킹 시스템이 특징인 보드게임 추적 앱. 경쟁적 플레이어 타겟.

**강점**:
- **게임별 ELO 랭킹** — 이 분야에서 가장 혁신적인 기능
- 성과 트렌드 차트 (시간별 실력 변화)
- Head-to-Head 비교
- BGG 데이터베이스 연동
- 무료 + 광고 없음

**약점**:
- iOS 중심 (Android 확장 중)
- 계정 필요
- 순위 기반 입력 미지원
- 웹 버전 없음
- ELO가 의미 있으려면 다수의 세션 필요

**통계 심층**:
- ELO 시스템: 게임별 독립 레이팅, 상대 실력 반영
- 트렌드 차트: 시간축 기반 레이팅 변화 그래프
- 비교: 두 플레이어 간 전적, 승률, 평균 점수

**시사점**: **ELO/스킬 레이팅은 장기적으로 매력적인 기능**. 달무티 맥락에서는 간소화된 버전(세션 간 평균 순위 트렌드, 상대 전적)이 적합. "게스트 플레이어가 나중에 기록을 claim" 패턴도 흥미로움.

## 5. 차별화 분석

### 5.1 달무티 점수판의 고유 강점 (이미 구축됨)

| # | 차별점 | 설명 | 경쟁앱 대비 |
|---|--------|------|-------------|
| 1 | **순위 드래그앤드롭 입력** | 플레이어를 순서대로 배치하면 점수 자동 계산. 수동 계산 오류 제거 | 모든 경쟁앱: 포인트 증감(+/-) 방식만 지원 |
| 2 | **달무티 전용 점수 로직** | `score = rank - 1`. 낮을수록 좋음. 네이티브 지원 | BG Stats도 수동 점수 입력 필요 |
| 3 | **라운드별 참가자 선택** | 특정 라운드에 불참한 플레이어를 null 처리 | 대부분의 앱에서 미지원 |
| 4 | **웹 기반, 설치 불필요** | URL 공유로 즉시 접근. 앱 스토어 불필요 | KeepTheScore만 웹 (but 범용) |
| 5 | **한국어 우선** | 모든 UI가 한국어. 한국 보드게임 문화 친화적 | 대부분 영어 우선, 한국어는 2차 언어 |
| 6 | **순위 뱃지 시각화** | 금/은/동 뱃지로 즉각적 순위 인식 | 어떤 경쟁앱에도 없음 |

### 5.2 해소해야 할 기능 격차

#### UX 개선 (사용자 관심 영역 1)

| 기능 | 현재 상태 | 우선순위 | 참고 경쟁앱 |
|------|----------|----------|-------------|
| **다크 모드** | 미지원 | HIGH | Score Anything, KeepTheScore, Scored |
| **테마/색상 커스터마이징** | 미지원 | MEDIUM | Score Anything (24색), KeepTheScore (로고/색상) |
| **점수 변경 애니메이션** | 미지원 | MEDIUM | KeepTheScore (카운트업), Scory (트랜지션) |
| **PWA (홈 화면 추가)** | 미지원 | HIGH | 네이티브 앱 접근성에 대응하는 핵심 수단 |

## 6. 전략적 추천사항

### Tier 1 — 즉시 구현 (핵심 차별화 강화)

1. **다크 모드** — Score Anything, Scored 참고. Tailwind CSS v4의 `dark:` 유틸리티로 구현 용이. 시스템 설정 연동 + 수동 토글.
2. **기본 통계 대시보드** — 플레이어별 승률, 평균 순위, 총 게임 수, 최고 순위. BG Stats의 통계를 달무티 맥락으로 축소.
3. **점수 추이 그래프** — 세션 내 라운드별 누적 점수 그래프 + 세션 간 성과 트렌드. Scored의 트렌드 차트 참고.
4. **달무티 역할명 표시** — 1위: 대달무티, 2위: 소달무티, 꼴찌: 대빈민, 꼴찌-1: 소빈민. 어떤 경쟁앱에도 없는 고유 기능.

### Tier 2 — 차기 구현 (경쟁 격차 해소)

5. **PWA manifest + 서비스 워커** — "홈 화면에 추가"로 네이티브 앱 수준 접근성.
6. **결과 공유** — 세션 요약을 텍스트/이미지로 카카오톡 등에 공유. KeepTheScore의 URL 공유 모델 참고.
7. **애니메이션** — 순위 변동 시 트랜지션, 점수 카운트업, 1등 축하 효과.
8. **라운드 타이머** — 하우스 룰(턴 제한 시간) 지원. Scory, Keep Score GK 참고.

### Tier 3 — 장기 구현 (프리미엄 기능)

9. **Head-to-Head 비교** — 두 플레이어 간 전적, 승률, 평균 점수 차이.
10. **간소화된 스킬 레이팅** — 세션 간 순위 기반 레이팅. Scored의 ELO 참고하되 간소화.
11. **커스텀 게임 규칙** — 점수 체계 변경 (예: 1등 보너스, 연승 보너스 등).
12. **다국어 지원** — 영어 추가로 해외 달무티 플레이어 확보.

### 하지 말아야 할 것

- **범용 점수 추적기로 확장 금지** — 시장 포화. 달무티 전문성이 강점
- **주사위/랜덤 선택기 추가 금지** — 달무티 게임플레이와 무관
- **계정/로그인 요구 금지** — 즉시 사용 가능한 모델이 핵심 장점
- **BGG 연동 금지** — 달무티 플레이어는 캐주얼 카드게임 사용자, BGG 헤비 유저가 아님

## 7. 경쟁 포지셔닝 맵

```
              게임 특화
                 ^
                 |
    Scored       |      달무티 점수판
  (BGG ELO,      |    (달무티 전용,
   게임별 통계)   |     순위 입력)
                 |
  범용 ----------+---------> 전문
                 |
  BG Stats,      |
  Scory,         |
  Score Anything,|
  Keep Score GK, |
  KeepTheScore   |
                 |
              범용 추적기
```

**달무티 점수판은 유일한 "전문 + 게임 특화" 영역을 차지**한다. 가장 가까운 플랫폼 경쟁자(KeepTheScore.com)는 범용이고, 가장 가까운 게임 특화 경쟁자(Scored)는 네이티브 전용이다.

## 8. 결론

달무티 점수판은 **방어 가능한 니치**를 확보하고 있다: 유일한 웹 기반 달무티 전용 점수 추적기로서 순위 드래그앤드롭 입력이라는 독보적 UX를 보유.

사용자가 관심을 표한 3개 영역에 대한 요약:

| 영역 | 현재 수준 | 경쟁 대비 | 우선 조치 |
|------|----------|-----------|-----------|
| **UX 개선** | 기본 반응형 | 다크 모드, 테마 부재 | 다크 모드 + PWA |
| **통계/분석** | 없음 | BG Stats, Scored 대비 열위 | 승률 + 점수 추이 그래프 |
| **게임플레이 확장** | 기본 순위 입력 | 역할명, 타이머 부재 | 달무티 역할명 표시 |

**성장 전략은 명확하다**: 달무티 전용 기능을 깊게 파고 (역할명, 통계, 규칙), UX 폴리시를 추가하되 (다크 모드, 애니메이션), 범용 점수 추적기로 확장하지 않는 것이 핵심이다.
